## 📋 프로젝트 개요

### 목적
개발자와 디자이너가 자신의 작업물과 생각을 공유할 수 있는 개인 블로그 플랫폼을 만든다.

### 타겟 사용자
- **1차**: 개발자, 디자이너 (20-35세)
- **2차**: 기술에 관심 있는 일반인

### 핵심 가치
- **간편함**: 복잡한 설정 없이 바로 글쓰기 시작
- **아름다움**: 코드와 이미지가 아름답게 표시되는 에디터
- **소통**: 댓글과 좋아요를 통한 독자와의 소통

## 🎯 주요 기능

### 1단계 (MVP - 2주)
1. **사용자 인증**
   - 이메일 회원가입/로그인
   - 구글 소셜 로그인

2. **블로그 글 관리**
   - 마크다운 에디터로 글 작성
   - 글 목록 조회
   - 글 상세 보기
   - 글 수정/삭제

3. **기본 프로필**
   - 프로필 이미지 업로드
   - 자기소개 작성

### 2단계 (추가 기능 - 2주)
4. **댓글 시스템**
   - 글에 댓글 작성
   - 댓글 답글 기능

5. **좋아요 & 북마크**
   - 글 좋아요 기능
   - 나중에 읽을 글 북마크

6. **검색 & 카테고리**
   - 글 제목/내용 검색
   - 태그별 글 분류

## 👥 유저스토리

### 핵심 사용자 여정
1. **방문자** → 블로그 글 둘러보기 → 관심 있는 글 발견
2. **회원가입** → 프로필 설정 → 첫 글 작성
3. **정기 방문** → 새 글 작성 → 댓글 확인 → 다른 사람 글에 댓글

### 주요 유저스토리
- **방문자로서** 회원가입 없이도 글을 읽고 싶다 **왜냐하면** 부담 없이 콘텐츠를 확인하고 싶기 때문이다
- **작성자로서** 마크다운으로 글을 쓰고 싶다 **왜냐하면** 코드와 텍스트를 아름답게 표현하고 싶기 때문이다
- **독자로서** 좋은 글에 좋아요를 누르고 싶다 **왜냐하면** 작성자에게 감사를 표현하고 싶기 때문이다

## 🎨 디자인 방향성

### 브랜딩
- **컨셉**: 미니멀하고 깔끔한 개발자 감성
- **컬러**: 다크모드 지원, 블루 계열 포인트 컬러
- **폰트**: 코드용 모노스페이스 폰트 + 본문용 산세리프

### 주요 화면
1. **메인 페이지**: 최신 글 목록 + 인기 글
2. **글 작성**: 마크다운 에디터 + 실시간 미리보기
3. **글 상세**: 깔끔한 읽기 화면 + 댓글
4. **프로필**: 작성자 정보 + 작성 글 목록

## 🔧 기술 요구사항

### 필수 기능
- **에디터**: 마크다운 지원, 코드 하이라이팅

- **반응형**: 모바일/태블릿/데스크톱 지원

- **이미지**: 자동 압축 및 최적화
- **캐싱**: 정적 콘텐츠 캐싱
